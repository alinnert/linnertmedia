Einstellungen

Farbschema

Aktuell wird nur das automatische Farbschema unterstützt, das sich an die aktuellen Systemeinstellungen richtet. Das ist eine Einschränkung von TailwindCSS.

Als App installieren

Wenn die Webseite als App installiert wird, kann sie über den Home-Screen, das Start-Menü o. ä. geöffnet werden.

Benachrichtigungen

Möchtest du sofort über neue Beiträge benachrichtigt werden, auch wenn du die Webseite gerade nicht geöffnet hast?

Offline-Modus

Im Offline-Modus kann die Seite auch ohne Internerverbindung aufgerufen werden.
